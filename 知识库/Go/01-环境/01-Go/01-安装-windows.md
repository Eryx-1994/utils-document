### Go - 配置环境变量

> `此电脑` -> `属性` -> `高级系统设置` -> `环境变量`

```
# 新建系统环境变量
GOROOT -> E:\IT_zhengqing\soft\soft-dev\Go\Go
GOPATH -> E:\IT_zhengqing\code\go-workspace

# 编辑PATH环境变量，新增
%GOROOT%\bin
```
