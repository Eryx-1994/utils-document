## Etherpad

项目地址：https://github.com/ether/etherpad-lite

```
# 运行体验：
docker run -d  --name etherpad  -p 9001:9001 -e 'ADMIN_PASSWORD=supersecret' etherpad/etherpad
```

可安装插件：
comments_page_short_edition
maadix
tables4
