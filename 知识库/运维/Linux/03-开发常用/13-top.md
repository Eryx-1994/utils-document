# top

```shell
# 查看进程占用资源 （退出：按q）  
top
# 按下M：结果按照内存占用从高到低排序  
# 按下P：结果按照CPU占用率从高到低排序  
# 按下1：当服务器含有多个CPU的时候，可以切换显示各个CPU的详细信息  
```

- zombie：僵尸进程个数
- PID：进程id
- USER：该进程对应的用户
- PR：优先级
- NI：用户进程空间内改变过优先级的进程占用CPU百分比
- VIRT：虚拟内存
- RES：常驻内存
- SHR：共享内存 -- 计算一个进程实际使用的内存 = 常驻内存 - 共享内存
- `s`：表示进程的状态，sleep，running
- `%CPU`：表示CPU的占用百分比
- `%MEM`：表示内存的占用百分比
- TIME：执行的时间
- `COMMAND`：进程的名称或者路径
