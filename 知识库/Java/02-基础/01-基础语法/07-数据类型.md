### 数据类型

#### 八种基本数据类型

1. 整型：byte、short、int、long
2. 字符型：char
3. 浮点型：float、double
4. 布尔型：boolean

| 基本数据类型 | 对应包装类 |
| :----------: | :--------: |
|     byte     |    Byte    |
|    short     |   Short    |
|     int      |  Integer   |
|     long     |    Long    |
|    float     |   Float    |
|    double    |   Double   |
|   boolean    |  Boolean   |
|     char     | Character  |
