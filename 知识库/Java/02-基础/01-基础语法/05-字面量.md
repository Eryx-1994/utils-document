### 字面量

字面量：被写下来的固定值

```java
public class Hello {
    
    public static void main(String[] args) {
        System.out.println(1);
        System.out.println(1.0);
        System.out.println("字符串");
        System.out.println(true);
        System.out.println(false);
        System.out.println("null");
    }

}
```
