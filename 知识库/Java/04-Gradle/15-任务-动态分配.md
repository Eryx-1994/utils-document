在循环中注册同一类型的多个任务

```
4.times { counter ->
    tasks.register("task$counter") {
        doLast {
            println "I'm task number $counter"
        }
    }
}

```

注册任务后，可通过API访问

> ex: 在运行时动态向任务添加依赖项

```
4.times { counter ->
    tasks.register("task$counter") {
        doLast {
            println "****** task $counter"
        }
    }
}
// task0强依赖于task2和task3(需在task0之前优先加载)
tasks.named('task0') {
    dependsOn('task2', 'task3')
}
```
