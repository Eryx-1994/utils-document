# 运算符

### 算术运算符

1. `+`：加法
2. `-`：减法
3. `*`：乘法
4. `/`：除法
5. `//`：取整除
6. `%`：取余
7. `**`：指数

```
print("1 + 1 = ", 1 + 1)
print("2 - 1 = ", 2 - 1)
print("3 * 3 = ", 3 * 3)
print("4 / 3 = ", 4 / 3)
print("5 // 3 = ", 5 // 3)
print("5 % 3 = ", 5 % 3)
print("2 ** 3 = ", 2**3)
```

### 赋值运算符

把 = 号右边的结果 赋给 左边的变量

```
num = 1 + 2 * 3
print(num)
```

### 复合运算符

```
num += 1
print("[加法赋值运算符] num += 1:  ", num)
num -= 1
print("[减法赋值运算符] num -= 1:  ", num)
```
